Why Sibyl-WAF?

Traditional WAFs rely on brittle regex rules. Sibyl thinks like an analyst.

AI-Driven Analysis

Powered by Gemini LLM, every HTTP request is evaluated for malicious intent using advanced natural language reasoning — not pattern matching.

Zero-Day Protection

No signature database to update. Sibyl understands attack semantics, catching novel exploits that traditional WAFs miss entirely.

Deterministic Gatekeeper

Crime Coefficient scoring (0-100) provides a clear, numeric verdict. ≥75 means instant termination. No ambiguity, no false negatives.

Distributed Architecture

Two microservices. Zero single point of failure. Full isolation.

COMPONENT A

Sibyl-Proxy

Home Server / Edge

  • ▹ Golang reverse proxy via net/http/httputil
  • ▹ Intercepts private routes from Nginx/Cloudflare
  • ▹ In-memory caching via sync.Map (300s TTL)
  • ▹ Strict context.WithTimeout (1500ms)
  • ▹ 2MB payload limit via io.LimitReader
COMPONENT B

Sibyl-Brain

Google Cloud Run

  • ▹ Cognitive reasoning engine (API Gateway)
  • ▹ Gemini API integration for intent analysis
  • ▹ Deterministic JSON response schema
  • ▹ Crime Coefficient scoring (0-100)
  • ▹ Static dashboard via //go:embed
